The content type group object enables related fields to be grouped together in the UI.

## Properties

Name

Type

Format

Description

id

string

A group identifier which is unique within the content type

name

object

[Localized value](/help-and-docs/apis/management-js/key-concepts/localization)

The friendly name given to the content type group

## Example

JSON

```
{
    "id": "mainContent",
    "name": {
        "en-GB": "Main content"
    }
}
```
