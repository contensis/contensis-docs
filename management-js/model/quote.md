The quote object represents a section of referenced text from an external source.

## Properties

Name

Type

Format

Description

text

string

The quote text

source

string

The source of the quote

## Example

This example shows a quote object.

JSON

```
{
    "text": "I'll be back...",
    "source": "The Terminator"
}
```
