A ContentTypeListOptions is a structure that is used to describe details for requesting content types.

Name

Type

Description

versionStatus

string

The version status, either *published* or *latest*. The default is *latest*.

dataFormat

string

The specific dataFormat, e.g. *entry*, *asset*. The default is all.
