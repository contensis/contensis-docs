The location object represents a geographical coordinate that specifies the position of a point on the Earth's surface.

## Properties

Name

Type

Format

Description

lat

number

The north-south position

lon

number

The east-west position

## Example

JSON

```
{
    "lat": 52.415789,
    "lon": -2.640774
}
```
