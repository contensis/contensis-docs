## API reference

## Delivery

The Contensis Delivery API is a platform agnostic API that focuses on delivering content created in content types and entries to your website or application.

-   [HTTP](https://www.contensis.com/help-and-docs/apis/delivery-http)
-   [Javascript](https://www.contensis.com/help-and-docs/apis/delivery-js)
-   [.Net](https://www.contensis.com/help-and-docs/apis/delivery-dotnet)

## Management

The Contensis Management API allows you to import and manage content within content types and entries. It is a HTTP API with JavaScript and .Net wrappers.

-   [HTTP](https://www.contensis.com/help-and-docs/apis/management-http)
-   [Javascript](https://www.contensis.com/help-and-docs/apis/management-js)
-   [.Net](https://www.contensis.com/help-and-docs/apis/management-dotnet)

## Image API

The Contensis Image API lets you manipulate and apply image transformations to images in Contensis and speed up image delivery.

-   [Image API](https://www.contensis.com/community/blog/developing-with-the-contensis-image-api)
