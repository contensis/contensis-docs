To update a webhook, you must be a System Administrator or a member of a role with Webhook permissions.

### Access the webhook

1.  Navigate to the Webhooks screen by selecting Settings in the sidebar, then choosing Webhooks from the menu.
2.  Click the name of the webhook to open it for editing, or select Edit from the more actions context menu.

### Update the name

To update the name, edit the Name field. This name is only used for identification.

### Update the description

The webhook description provides summary information about what the webhook is used for and is displayed in the webhook listing. You can update it by editing the Description field.
