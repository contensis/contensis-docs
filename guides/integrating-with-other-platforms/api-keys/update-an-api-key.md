If you want to update the name or description of an API key, you can do so by following these steps:

1.  Press the **Settings** button in the sidebar. The *Settings* menu will open, revealing a number of options.
2.  Select **API keys** to open the *API Keys* list.
3.  Find the API key in the list that you want to update, and press **Edit**. The *Edit API key* screen will open. Make your changes to the name or description.
4.  Press **Update API Key** to save your changes. The *API Keys* list will update with the changes you've made.
