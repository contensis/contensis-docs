If you no longer use the application that was accessing your content through a particular API key, you can delete the API key permanently.

1.  Press the **Settings** button in the sidebar. The *Settings* menu will open, revealing a number of options.
2.  Select **API keys** to open the *API Keys* list.
3.  Find the API Key you want to delete in the list, and press the **Action** button denoted by the triple dot icon.
4.  Select **Delete** from the menu. The delete confirmation screen will open.
5.  Confirm the deletion of the API key by pressing **Delete**. The API key will be permanently removed.

**Warning:** An application or website using this API key will expect it to exist to access its content. Deleting the API key is irreversible.
