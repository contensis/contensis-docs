The project listing screen provides an overview of all projects in your Contensis environment. This article explains how to access, filter, and organise your projects.

## Accessing your projects

1.  Select the square project avatar button in the sidebar
2.  View the complete list of available projects

## Filtering projects

If your environment contains numerous projects, you can easily narrow down the list:

1.  Locate the filter bar at the top of the project listing
2.  Type a relevant word or phrase
3.  The project list will automatically update to show only matching projects

## Managing favourite projects

The favourites feature helps you quickly access frequently-used projects by placing them at the top of your project listing.

### Adding a project to favourites

1.  Open the project listing view
2.  Locate the project you wish to favourite
3.  Press the unfilled star button next to the project name
4.  The project will move to the top section of your listing

### Removing a project from favourites

1.  Open the project listing view
2.  Find the favourited project (located at the top of the list)
3.  Press the filled star button next to the project name
4.  The project will return to the main alphabetical listing
