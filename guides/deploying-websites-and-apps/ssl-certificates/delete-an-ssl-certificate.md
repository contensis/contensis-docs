To remove an SSL certificate you'll need to be a System Administrator or a member of the SSL Certificate Administrators group.

1.  Navigate to SSL Certificates screen from the settings menu in the sidebar
2.  With the certificates list displayed, open the context menu for the certificate.
3.  Select **Delete certificate**.
4.  Confirm in the **Delete certificate?** window.
5.  On confirmation, the certificate is removed and a notification is displayed confirming the deletion.
