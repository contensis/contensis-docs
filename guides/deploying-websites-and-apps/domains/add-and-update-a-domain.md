## Add a new custom domain

-   Navigate to the Domain listing screen by pressing **Settings** in the sidebar and selecting **Domains** from the menu.
-   Select **New domain** from the toolbar. The *Add a domain* window will open.
-   Enter the domain name that you want to add, either in the format `www.yourdomain.com` or `docs.yourdomain.com`
-   Press **Add domain** to add the domain to your project. A notification will be displayed confirming that the Domain has been added.

When a domain is added we carry out a DNS verification to validate if you have pointed your DNS to Contensis. If you've not done this then you will see a failed DNS status for the newly added domain.

## Update a domain

If you need to update the domain name in your project follow these steps.

-   Locate the domain in the domains listing.
-   From the actions menu for the domain choose **Edit domain** from the menu, an update domain window will be displayed. -
-   Make the changes to your domain and press **Update domain**. A notification will be displayed confirming that the Domain has been updated.
