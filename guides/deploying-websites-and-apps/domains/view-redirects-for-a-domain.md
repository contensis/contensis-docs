The domains listing screen will display how many redirects have been configured for the domain. To view the redirects for the domain, you’ll need to be a System administrator or a member of a role with redirect permissions.

-   Locate the domain in the domains listing.
-   Select the **more actions** button for the domain, indicated by the triple dot icon.
-   Press **View redirects** from the menu, you’ll be navigated to the Redirects screen with a filter for the domain automatically applied.
