Page last updated 28 May 2024

Renderers can be created and managed by a System Administrator or another user that is a member of a role with sufficient permission.

The renderer resource supports the following actions.

Action

Description

Create

The ability to create a new renderer. Usually combined with the Update action.

Update

The ability to save changes to a new or existing renderer.

Delete

Allow a member of a role to delete a renderer.

View

Allow members of a role to see renderers, but not make any changes.
