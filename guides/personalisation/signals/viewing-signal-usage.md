Signals define the characteristics of site visitors that audiences use to segment and target content. Signals are assigned to audiences, and you can view where a signal is used through the Signal usage panel in the Signal editor.

## How to view signal usage

Follow these steps to see where a signal is used:

1.  Navigate to the Signals screen. Select **Personalisation** from the sidebar, then choose **Signals** from the menu.
2.  In the signal listing, select the signal you want to review.
3.  In the vertical navigation of the editor, press **Usage**. This opens the usage panel, which displays any related usage.

## What you’ll see

The usage panel lists:

-   Audiences – All audiences that reference the signal.

You can navigate directly to any audience by selecting it in the list.
