Signals can be created and managed by a Systems Administrator or another user that is a member of a [role](https://www.contensis.com/help-and-docs/guides/governing-content/roles-and-permissions/assign-and-remove-role-permissions) with sufficient permission.

The Signal resource supports the following actions.

Signals actions

Action

Description

View

Allows members of a role to see signals from the Personalisation menu in the sidebar.

Create

The ability to create a new Signal. Usually combined with the update action.

Update

The ability to save changes to a new or existing Signal.

Publish

Allows members of a role to publish changes to a Signal.

Delete

Allows members of a role to delete existing Signals.
