Custom attributes can be created and managed by a Systems Administrator or another user that is a member of a [role](https://www.contensis.com/help-and-docs/guides/governing-content/roles-and-permissions/assign-and-remove-role-permissions) with sufficient permission.

The custom attribute resource supports the following actions.

Custom attribute actions

Action

Description

View

Allows members of a role to see custom attributes from the Personalisation menu in the sidebar.

Create

The ability to create a new custom attribute. Usually combined with the update action.

Update

The ability to save changes to a new or existing custom attribute.

Delete

Allows members of a role to delete existing custom attributes.
