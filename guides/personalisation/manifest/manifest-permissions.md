The personalisation Manifest can be created and managed by a Systems Administrator or another user that is a member of a [role](/help-and-docs/guides/governing-content/roles-and-permissions/assign-and-remove-role-permissions) with sufficient permission.

The Manifest resource supports the following actions.

Manifest actions

Action

Description

View

Allows members of a role to see the Manifest menu item from the Personalisation menu in the sidebar.

Publish

Allows members of a role to publish changes to the Manifest.
