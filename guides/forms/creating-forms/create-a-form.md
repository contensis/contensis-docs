4.  Creating forms

To create or update an existing for you'll need to be a member of a role with Form Create and Update permission. Follow these steps to create a form:

1.  Press **Forms** in the sidebar. The *Forms* drawer will open, revealing a number of options.
2.  Select **Forms** to open the *Forms listing* screen.
3.  Press **New form**. AÂ *Create new form* window will be displayed.
4.  Give your form a name, such as 'Contact us'.
5.  The API name will automatically be created and sanitised from the form name you enter.
6.  You can add an optional description to help you understand the forms use in the listing.
7.  Press the **Create** button to create your new form. The *Form builder* will open so you can start [adding fields](/help-and-docs/guides/forms/creating-forms/add-remove-fields-in-a-form) to your form.
