Forms can be created and managed by a Systems Administrator or another user that is a member of a [role](https://www.contensis.com/help-and-docs/guides/governing-content/roles-and-permissions/assign-and-remove-role-permissions) with sufficient permissions.

The form resource supports the following actions:

Action

Description

Create

The ability to create a new form. Usually combined with the update action.

Update

The ability to save changes to a new or existing form.

Delete

Allows a member of a role to delete a form.
