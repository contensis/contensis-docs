Users are created and managed at an organisational level in Contensis.

Users can be created within the Contensis interface, through the Security API or can be authenticated using your corporate identity provider by configuring Single Sign On.

### Types

-   **Contensis** – A user created through the Contensis interface or through the Security API.
-   **External** – A user that is generated through a corporate identify provider when Single Sign On is configured.
-   **API Key user** – A special user that is created when an API key is created that can be granted permissions through assignment to a role or Contensis group.
