Only System Administrators can reset a user's password.

With the *User listing* screen displayed:

1.  Locate the user whose password you want to reset.
2.  Press the **Actions** button in the listing. A context menu will be displayed.
3.  Select **Reset password** from the menu. A *Reset password* window will appear.
4.  Set a new password for the user and whether you want the password to be reset at the next login.
5.  Confirm the password change by pressing **Reset password**.
