Only System Administrators can delete users from an organisation.

With the *User listing* screen displayed:

1.  Locate the user you want to delete
2.  Press the **Actions** button in the listing. A context menu will be displayed.
3.  Select **Delete** from the menu. A *Delete user confirmation* window will appear.
4.  Confirm the deletion of the user by pressing the **Delete** button. The user will be permanently removed from the organisation.
