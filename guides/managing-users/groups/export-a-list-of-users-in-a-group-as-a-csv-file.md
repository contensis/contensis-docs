The Group listing screen allows you to export a list of users belonging to a specific group in CSV format. This feature is useful for creating reports, auditing user access, or managing your user base.

## Exporting users to CSV

1.  Navigate to the Group listing screen
2.  Locate the group you want to export users from
3.  Press the **Actions** button in the listing. A context menu will appear with several options
4.  Select **Export users as CSV** from the menu. Your browser will either prompt you to save the file or automatically download it to your downloads folder

## CSV file contents

The exported CSV file contains comprehensive user information with the following fields:

-   firstName
-   lastName
-   email
-   userName
-   created
-   modified
-   language
-   lastLogin
-   passwordChanged

## Working with the exported data

Once downloaded, you can:

-   Open the CSV file in spreadsheet applications like Excel or Google Sheets.
-   Sort and filter user data as needed.
-   Analyse user activity based on lastLogin and passwordChanged dates.
-   Create custom reports based on the exported information.
