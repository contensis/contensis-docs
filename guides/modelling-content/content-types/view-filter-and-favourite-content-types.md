Content types are project specific. You can view content types by navigating to the *Content types* listing. Access to this screen is restricted to users in the *Content Type Administrators* security group.

## Content types listing screen

1.  Press the **Content modelling** button in the sidebar. The *Content modelling* drawer will open, revealing a number of options.
2.  Select **Content types** to open the *Content types* listing screen.

## Filter content types

If you have a large number of content types in your project, you may find it easier to search using the filter bar.

-   Type a word in the filter bar to narrow down the content types displayed.

## Favourite/unfavourite content types

If you find yourself regularly managing the same content types amongst a large content model, you may find it helpful to favourite them. Favourites are displayed at the top of the listing

### Add to favourites

With the content type listing open, press the unfilled star button on the content type. The content type will be added to your favourites and raised to the top of the listing.

### Remove from favourites

With the content type listing open, press the filled star button on the content type. The content type will be removed from your favourites and returned to the main listing.
