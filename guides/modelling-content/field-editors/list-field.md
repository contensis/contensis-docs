The list field editor allows authors to select from a list of items. You specify these items in the *List values* text box found in the **Editor** tab of the *Field Settings* panel.

The type of list displayed will depend on whether you allow multiple selection. You can choose whether to show single selection lists as a radio button list or as a dropdown. Checkbox lists are always used when multiple selection is set.

## Settings

Setting name

Summary

[Name](https://www.contensis.com/help-and-docs/user-guides/content-modelling/field-editors/field-settings)

A text label to identify the field in an entry.

[Field ID](https://www.contensis.com/help-and-docs/user-guides/content-modelling/field-editors/field-settings)

A sanitised name to be used by the API.

[Allow multiple items to be selected](https://www.contensis.com/help-and-docs/user-guides/content-modelling/field-editors/field-settings)

Allows an author to select multiple items in a list.

[Read only](https://www.contensis.com/help-and-docs/user-guides/content-modelling/field-editors/field-settings)

Prevents authors from editing the field in the entry editor.

### List values

Add items to the list field by typing a value and press the add button. List items can be ordered using drag and drop.
