Components can be created and managed by a Systems Administrator or another user that is a member of a [role](/help-and-docs/guides/governing-content/roles-and-permissions/assign-and-remove-role-permissions) with sufficient permissions.

The component resource supports the following actions:

Action

Description

Create

The ability to create a new component. Usually combined with the update action.

Update

The ability to save changes to a new or existing component.

Delete

Allows a member of a role to delete a component.

Publish

Allows a member of a role to publish changes made to a component.
