With a content type open for editing:

1.  Select the field you would like to make repeatable.
2.  In the *field settings panel*, set the *Allow multiple items to be added* toggle to **Yes**.
3.  You can set the button text label in the textbox that is displayed to let the user know what they are adding. For example, *Add new contributor* or *Add new opening time*.
4.  Click **Save**, followed by **Publish** to confirm your changes.

## Supported fields

The following fields can be set to repeatable when [creating](https://www.contensis.com/help-and-docs/user-guides/content-modelling/content-types/create-a-content-type) or updating a [content type:](https://www.contensis.com/help-and-docs/user-guides/content-modelling/content-types/content-types-overview)

-   Text
-   Markup
-   Quote
-   Number
-   Location
-   Date/time
-   Components
