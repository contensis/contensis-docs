Contensis users can collaborate on entries by starting conversations or responding to existing discussions on entries they can view.

## All users

The following actions and behaviour apply to all users.

Action

Description

Create

All users can comment on entries they can view.

Update

An author can update a comment for two minutes after posting their comment. An *Edited* status is displayed when an author updates a comment.

Reply

An author can reply to any conversation on entries they can view.

Delete comment

An author can delete any comment they have created.

Delete conversation

An author can delete any conversation they have started.

Resolve/Unresolve

The creator of a conversation or a System Administrator can resolve/unresolve a conversation.

## Role actions

Action

Description

Delete any comment

A member of a role with this action can delete any comment on any entry they can view.
