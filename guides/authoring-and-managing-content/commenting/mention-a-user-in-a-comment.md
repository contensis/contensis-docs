You can use the @mention feature to notify a specific user in a comment. This will ensure that the mentioned user sees the comment in their comment dashboard.

Follow these steps to @mention a user in a comment:

1.  With an entry open for editing or review, start a new conversation or reply to an existing thread.
2.  When typing your message, type `@` followed by the name of the person you want to mention.
3.  As you type, a list of suggestions will appear. Select the appropriate user from the provided list.
4.  Once you have finished composing your comment, press **Add Conversation** or **Reply**, depending on the action you are performing.
