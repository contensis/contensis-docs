To update a view to account for changes in the search criteria or columns displayed, follow these steps.

1.  Choose a view from the views listing. The selected view will be displayed.
2.  Update your search criteria using the basic search filters, or update the ZenQL query.
3.  Optionally, customise the columns displayed using [Manage columns](/help-and-docs/user-guides/authoring/entries/change-the-view-of-the-entry-listing).
4.  When a change to the view has been made, the view name is updated with (edited). Press **Update view** in the top toolbar to save your changes.
5.  A notification will be displayed confirming that the view has been updated.

**Note:** If you choose a view that you don't have permission to edit, you'll see the option to *Save as a new view* instead.
