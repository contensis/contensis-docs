Once an entry has been deleted and moved to the recycle bin, it will remain in the recycle bin for 90 days before being permanently deleted. In the event that you would like to permanently delete an entry before this period is reached you can choose to permanently delete the entry manually.

You'll need to be a member of a role with the delete permission for the content type to be able to see and permanently delete the entry from the project.

1.  Press the **Settings** button in the sidebar. The *Settings* menu will open, revealing a number of options.
2.  Select **Recycle bin** to open the recycle bin listing.
3.  Locate the entry in the listing you want to permenantly delete.
4.  Press the **Actions** button for the entry, as indicated by the triple dot icon. A dropdown menu will be displayed.
5.  Select **Permanently delete entry** in the menu. A *Permanently delete entry confirmation* window will appear.
6.  Press the **Permanently delete** button to confirm the action. A notification will be displayed confirming the permanent deletion**.**
