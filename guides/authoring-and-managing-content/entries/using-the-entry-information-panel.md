The *Entry information* panel provides an overview of entry details and user activity. It can be opened from the *Entry editor* toolbar or within the entry listing.

## What does it show?

The *Entry information* panel provides detail about the entry, including:

**Label**

**Explanation**

Current status

The stage of the workflow that the entry is currently in.

Published

Specified if the entry is currently published.

Created

The date and time that the entry was originally created

Last updated

The date and time that the entry was last updated.

Language

The language of the entry.

Content owner

The content owner will be the original creator of the entry, unless the content owner has been changed.

Content type

The content type that the entry is based upon.

Entry references

The number of entries that currently reference this entry. These relationships can be seen in the [entry usage](/help-and-docs/user-guides/authoring/entries/entry-usage) panel.

Entry ID

The unique identifier for this entry, sometimes referred to as the entry GUID.

It also provides a summary of user activity for each version of the entry. You will be able to see workflow actions that have taken place, along with time and the user who worked on the entry.

## Awaiting approval and declined workflow states

If an entry is in the *Declined* or *Awaiting approval* workflow state, when the entry is opened the activity panel is automatically displayed with the submission or declined message.
