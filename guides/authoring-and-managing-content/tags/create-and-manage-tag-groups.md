Tag groups organise related tags under a shared category, making your tagging system easier to manage. This article explains how to create, edit, and delete tag groups in Contensis.

## Create a new tag group

1.  Navigate to the *Tags listing* screen by selecting the **Content modelling** button in the sidebar.
2.  Select **Tags** from the menu to open the *Tag listing* screen.
3.  In the search panel, press the **Add tag group** (+).
4.  Enter a group Name.
5.  The API name is auto-generated from the name, but you can edit it if needed.
6.  If required, you can add an optional description up to 500 characters long.
7.  Enter a comma-seperated list of tags you want to include in the group.
8.  Click **Create** to finish. The tag group will be created, along with any new tags listed.

## Update a tag group

1.  In the tag group list, click the **More actions** button (⋮) next to the group.
2.  Select **Edit** from the menu.
3.  Update the name or description as needed.
4.  Press **Save** to apply your changes.

## Delete a tag group

-   From the tag group list, press the **More actions** button (⋮) next to the group.
-   Select **Delete** from the menu.
-   If the group is in use, a message will appear and deletion will be prevented.
-   If the group is not in use, you'll be prompted confirm deletion.
