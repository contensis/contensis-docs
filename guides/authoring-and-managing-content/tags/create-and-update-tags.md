Tags help you categorise and filter content in Contensis. This article explains how to create new tags within a tag group, and how to edit existing tags when needed.

You’ll need the appropriate permissions to create or update tags.

## Creating tags

Follow these steps to create new tags and assign them to a tag group:

1.  Navigate to the *Tags listing* screen by selecting the **Content modelling** button in the sidebar.
2.  Select **Tags** from the menu to open the *Tags listing* screen.
3.  Click the **New tags** button in the toolbar. A *New tags* window will appear.
4.  Enter one or more tag names (you can separate multiple tags with commas).
5.  Choose a **tag group** from the dropdown list.
6.  Press **Add tag(s)**.
7.  A confirmation message will be displayed.
8.  The new tags will be added to the selected tag group and shown in the list.

## Editing tags

You can update the label of an existing tag at any time:

1.  From the *Tags listing* screen, find the tag you want to update.
2.  Click the **More actions** button (**⋮**) next to the tag.
3.  Select **Edit** from the menu.
4.  Enter the new tag label in the editor.
5.  Press **Save** to update the tag.
6.  A confirmation will appear, and the updated tag label will be applied wherever the tag is used.

## Translate tags

If you have a multilingual project, you can also update the labels for other available languages from the same edit screen. This ensures tags are clearly understood across different language versions of your content. You can also [bulk translate tags](https://www.contensis.com/help-and-docs/guides/authoring-and-managing-content/tags/bulk-translate-tags-for-multilingual-content) using the translation editor.
