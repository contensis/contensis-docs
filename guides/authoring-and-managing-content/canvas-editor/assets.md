### Common uses

-   Add a downloadable PDF manual displayed as a card
-   Insert an image or audio file into your content
-   Embed a presentation or slide deck for reference
-   Add a product specification sheet in a support article
-   Insert a branding pack or logo download into a media kit page

### Add an asset

1.  Type `/asset` and press Enter.
2.  A picker window will appear where you can choose the asset you want to insert.

### Edit an asset

-   With the asset block focused, press the swap button in the toolbar to replace it with another asset.

### Delete an asset

You can remove an asset block in several ways:

-   Press Delete on the block toolbar.
-   Press Del or Backspace when the block is focused.
-   Open the actions menu by pressing Command/Control + / and select Delete.
