An access token is a read-only environment token to retrieve published content of any project through the Delivery API.

To find the access token for your environment, you'll need to be a System Administrator.

1.  Press the **Settings** cog in the sidebar.
2.  Select **Access token** from the menu. The access token for your environment will be displayed.
3.  Press the **Copy to clipboard** button to use the access token in your project.
